一、环境升级

本次升级本地环境，同时重新选择适配的主题，环境如下：

  • node.js v14.16.0
  • npm 6.14.11
  • hexo 5.4.0
  • hexo-cli 4.2.0

新主题下载地址：https://github.com/fluid-dev/hexo-theme-fluid

二、图片加载适配

本次修改主题直接使用hexo自带的图片语法

  1. 配置文件_config.yml下的属性post_asset_folder: true
  2. 使用语法![image_1.png](image_1.png)语法加载

或者直接使用img标签
